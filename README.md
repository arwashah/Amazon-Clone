# Amazon Clone 🛒

A static front-end clone of Amazon's website, built purely with **HTML** and **CSS** — no JavaScript or backend functionality involved. This project was created to practice and demonstrate skills in layout design, responsive UI, and replicating real-world website structures.

## 🔍 Features
- Amazon-style navigation bar (logo, search bar, account/orders, cart icon)
- Hero banner/carousel section
- Product grid layout with cards
- Footer replicating Amazon's structure
- Fully responsive design using CSS (Flexbox/Grid)

## 🛠️ Built With
- HTML5
- CSS3

## 📌 Note
This is a **UI/UX clone for educational purposes only** — it does not include any real functionality (cart system, payments, backend, etc.) and is not affiliated with Amazon in any way.
